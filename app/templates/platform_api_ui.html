Guia rapido

A ordem recomendada para testar tudo no navegador.

  1. Testar GET /health.
  2. Criar a conta operacional.
  3. Configurar Twilio.
  4. Configurar OpenAI.
  5. Configurar e-mail, se quiser fallback.
  6. Gerar o token.
  7. Enviar um lote.
  8. Consultar o lote ate o resultado final.
